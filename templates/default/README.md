# Default Template

This is the default SPEAKMAN.AI branded document template.

## Contents

| File | Purpose |
|---|---|
| `logo.png` | Header logo — rendered in the document header bar |

## Customising

To use your own brand, copy this directory to `templates/corporate/` and replace `logo.png` with your own logo (PNG, recommended height: 36px).

Then pass `--template templates/corporate` to `doc_writer.py`, or select `[corporate]` when the skill prompts you.
